Instructions

This is the interface for our implementation of a hands-free driving system. Listed Below are some functions that you can use.

  • Enable hands-free: H
  • Steer: A and S keys
  • Brake: S
  • Accelerate: W

Scenarios

Press the accompanying key to demonstrate the scenario.

  1. System fully operational: Ideal use case for such a system
  2. System failure: System relinquishes control to the driver
  3. Driver inattentive: System detects driver is not engaged and issues warnings
  4. Driver override: Driver takes control from the system
