Instructions

This is the interface for our implementation of a hands-free driving system. Listed Below are some functions that you can use.

  • Enable hands-free: H
  • Steer: A and D keys
  • Brake: S
  • Accelerate: W

Scenarios

Press the accompanying key to demonstrate the scenario.

  1. System fully operational:
    Ideal use case for such a system. The system steers the vehicle within the lane.
  2. System failure:
    System relinquishes control to the driver in the event of a failure.
  3. Driver inattentive:
    System detects driver is not engaged and issues warnings to them. The driver must touch the steering wheel to let the vehicle know they are there.
  4. Driver override:
    Driver takes control from the system. The driver decelerates to disengage the system. While the system is engaged pressing the accelerator, brake pedal, or steering will disengage the system as well.
